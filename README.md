## Projetos


### 🧬 Algoritmo Genético — Alocação de Salas
**Disciplina:** Tópicos Especiais em IA  
**Contexto:** Resolver o problema de alocação de salas de aula usando algoritmos genéticos como técnica de otimização.

O desafio foi modelar o problema de alocação como um problema de otimização combinatória — definindo representação do indivíduo, função de fitness, seleção, crossover e mutação. Construí também uma interface simples pra visualizar os resultados gerados pelo algoritmo.
